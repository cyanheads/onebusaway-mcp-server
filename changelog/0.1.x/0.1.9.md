# 0.1.9 — 2026-06-12

## Added

- **MCPB bundle cleaner** (`scripts/clean-mcpb.ts`) — runs after `mcpb pack` (wired into the `bundle` script) to run `mcpb clean` and strip dependency-shipped agent docs (`skills/`, `.claude/`, `.agents/`, stray `SKILL.md`) nested under `node_modules/` that root-anchored `.mcpbignore` patterns cannot reach.
- **Docker `HEALTHCHECK`** — bun-native `fetch` against `/healthz` (the slim image ships no `curl`/`wget`); `--interval=30s --timeout=5s --start-period=10s --retries=3`.
- **Docker `org.opencontainers.image.version` label** — populated from a new `APP_VERSION` build arg.

## Changed

- **Server identity** — `createApp()` now sets explicit `name` and `title` of `onebusaway-mcp-server`; `.codex-plugin/plugin.json` `interface.displayName` follows (`@cyanheads/onebusaway-mcp-server` → `onebusaway-mcp-server`), so every surface shows the hyphenated repo name rather than the npm scope.
- **`.mcpbignore`** — patterns are now root-anchored (leading `/`); adds `/.agents/` and `/skills/`.
- **Synced skills and vendored tooling** — `skills/**` re-synced to framework `^0.10.6`; new `skills/techniques/`; `scripts/check-framework-antipatterns.ts` and `scripts/lint-packaging.ts` updated from the framework.

## Dependencies
